# NestJS Starter Kit with TypeORM, PostgreSQL (running in Docker), and Jwt Auth

## Running the app

Change `.env-sample` file to `.env`, then set your environment variables.

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

$ npm run test:watch

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
